# Fullstack Todolist
Made this as a capstone project to see how apps are build using MERN, how database and frontend are operated together. 

## Tech Stack
**Frontend**  -> React + Tailwindcss 

**Backend** -> Node + Express

**Database** -> MongoDB

## Run this in your system
### Prerequisites

- MongoDB community server 
- Node.js 

Clone the repo
```
git clone https://github.com/ujjwal-py/Todolist-MERN.git
cd Todolist-MERN
```
Run the backned and frontend servers in separate terminals 
```
cd ./frontend
npm install
npm run dev

cd ./backend
npm install 
npm start
```

Set up your .env from .env.example 
```
MONGO_URI= # your monogo db connection url here
PORT= # port number where you want to run the backend
```
